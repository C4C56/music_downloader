# music_downloader
used for music download

KuGou music crawler 

可选参数
--search_type {music,singer}       default='music'
--page                             default='ALL'
--index                            default='ALL'
--quality {normal,high,best}       default='normal'

必传参数
store_folder                     
key_word

使用方法

python main.py key_word store_folder --search_type={} --page={} --index={} --quality={}
